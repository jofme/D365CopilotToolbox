# D365 Copilot Toolbox
A collection of extensions and integrations that bring Copilot Studio-built agents into D365 Finance & Operations.
By the community, for the community!

Inspired by the https://github.com/ameyer505/D365FOAdminToolkit project.
Initial idea by Lone Gru (lone.gru@arineo.com)

# Contact and Contribution
Want to contribute? We'd **LOVE** to have you! 

There are several ways to contribute:
 - Add new features
 - Report bugs
 - Request new features
 - Documentation

Feel free to reach out to copilot@erpilots.com for any questions or feature suggestions.
